# GUESS-THE-NUMBER
A beginner friendly Python Game where you guess the number with difficulty levels and hints.

------------------------------ GUESS THE NUMBER (Python Game) ------------------------------

A Simple console-based number guessing game written in Python.

------------------------------ FEATURES ------------------------------

- Random Number between 1 and 100
- 3 Difficulty Levels:
  - Easy -> 7 attempts
  - Medium -> 5 attempts
  - Hard -> 3 attempts
- Hints for "TOO HIGH" or "TOO LOW"
- Clean User experience with replay option

------------------------------ HOW TO PLAY ------------------------------

1. Run the script:
   python Guess_The_Number.py
2. Choose a difficulty level.
3. Keep guessing until you find the correct number or run out of attempts.
4. You can replay the game after finishing a round.
